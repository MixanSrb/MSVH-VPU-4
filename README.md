# MSVH-VPU-4

A 4-bit virtual CPU simulator written in C++.

## Overview

MSVH-VPU-4 is a minimal CPU emulator that simulates a simple 4-bit processor with registers, an ALU, flags, and basic control flow.

It supports a small assembly-like instruction set and executes programs using a program counter.

## Features

- 4-bit architecture (0–15 values)
- Registers: A, B
- ALU operations: ADD, SUB, MUL, DIV
- Flags: Z (zero), C (carry)
- Control flow: JMP, JZ, JNZ
- Program execution via instruction list

## Purpose

Built as a learning project to understand CPU design, instruction execution, and low-level computing concepts.

## Naming

MSVH-VPU-4 stands for **MixanSrb Virtual Hardware (MSVH) - Virtual Processing Unit (VPU) - 4 bit**.
